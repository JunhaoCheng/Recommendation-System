# 项目名称  
基于Apriori算法的音乐推荐系统  

# 项目简介  
该系统由服务端和微信客户端两部分组成，服务端基于Apriori算法。  
训练关联规则模型（Apriori）以查找过去一段时间用户所听音乐类  
型的最相关项.该算法将大多数用户的产品偏好关联起来，并可用于生  
成音乐类型推荐。
